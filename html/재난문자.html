■ 문의 :  재난문자 수신 불가 
■ 확인 및 처리내용 : 
ㅇ 재난문자란 : 각종 재난 발생시 신속한 대피를 위해 국민안전처에서 
   휴대폰으로 보내는 긴급 문자메시지
ㅇ 주요 내용 
-  지진으로 인한 '긴급재난문자'는 기상청에서 직접 발송하도록 변경되었음. 16.11.21 ~ 현재
   ( 그외 기타 재난문장의 경우 '국민안전처'에서 담당)
- 삼성폰의 경우 : 갤럭시 S4 부터, 노트3부터 사용가능 
- 아이폰은 아이폰 5 이상부터 가능 
- 넥서스 5 는 재난문자 미지원 
- LG :  2013.1.1 이후 출시된 단말(G프로)부터는 의무적으로 기능 탑재
- 기타 : 베가레이서2 이후 단말부터 지원 / 비와이(화웨이) 가능
※ OPENMODEL2 의 경우, 당사에서 검증하지 않은 단말기로 단말기에서 
 재난문자 수신여부를 확인 불가
※ 당사에서 출시한 검증된 모델의 경우는 선불폰(LTE단말기+LTE선불요금제+LTE유심 사용시)도 
 재난문자 수신이 가능
※ 가능한 단말이라도 단말기 설정상태에 따라 미수신될수 있음
 (받고싶지 않을경우에도 해당메뉴 활용) 
안드로이드 : 문자메세지 -> 설정 -> 긴급알림설정 -> 긴급재난문자 비활성화 or 활성화 
 ( 기종별 진입메뉴 차이 있으나 대부분 유사함 ) 
아이폰 : 설정 -> 알림 ->재난문자 발송 -> 수신설정
