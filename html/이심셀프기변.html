  사례1 . 두 단말 간 이동 기변 
 
  ※모든 절차는 New단말기에서 시작함
  ※진행중 네트워크가 끊어지기 떄문에 NewPhone에서 와이파이 연결이 
  되어 있어야 오류가 발생되지 않음 
  ( 와이파이 설정 안내 메시지가 팝업됨 )
 
  ※보내기 단말은 화면잠금이 안된 단말기 인경우 보내기 자체가 
  진행되지 않기 때문에 먼저 잠금 화면 설정부터 필요함/화면 잠금 
  설정된 단말기에서 잠금 해제후 진행이 가능함 (다소 맹점이라고 함) 
 
  New단말 설정-연결-sim관리자-esim추가-다른 기기에서 심이동
 
  - old단말을 new단말로 근접시킴-old단말 화면에 "이동"버튼 터치 -
   new단말에 인증번호 팝업됨 - 펍업된 인증 번호를 old단말기 
   인증번호칸에 입력 
  
  - new단말에서 기변할 회선 선택 화면이 보임 (ex. a번호 ,b번호) -
   선택하면 old단말기에서 선택한 기기로 이동할까요? 물어보는 창이 
    나오고 거기서 확인 터치
 
  - new단말기에서 Esim셀프 기기변경 서비스 안내 화면이 나오고 확인 
  터치후   체크창 안내대로 체크(이용약관등 확인됨)
 
 
  - 인증수단 선택 창 ( 핸드폰 /신용카드 인증 선택) /
   ※old단말기가 멀티 회선 상태라면 이동하려는 회선 번호를 입력해 
   줘야 정상적인 본인인증 처리가 됨
 
  - 본인인증 성공후 Esim추가 화면으로 자동전환됨 - 최종 처리후 
  new폰에서 sim관리자 화면으로 자동 전환됨 - old폰 확인시
      (개통이 필요합니다 문구 팝업됨 / 유심정보가 이동했음으로 ) 
 /진행후 new폰 심관리자에서 번호정보가 보이지 않을 경우 1-2분정도 
  기다리거나 오류가 발생된 경우 설정 sim관리자로 재접근해 ESIM추가
  를 다시 누르면 kos에 이미 배정되어 있는 정보가 내려감
