■ 증상: 에그(IML520) 단말기 USB 테더링 불가 
- PC에 USB 케이블 연결하여 테더링 시도 시 안됨

■ 확인 및 조치 사항: 
- 하기 USB 테더링 절차 확인
- WCM 주소 192.168.1.1 접속 하여 RNDIS 사용유무 '사용'으로 설정 필요하나 
- PC에 WCM 사이트 접속 자체 안됨 / 웹페이지를 표시 할 수 없다고만 팝업됨
※ 에그 WIFI 연결하여 PC 內 타사이트는 접속 가능하나 WCM 사이트만 접속안됨 
- 타기기(핸드폰)로 해당 에그 WIFI 연결시에는 WCM 정상접속됨
- WCM 로그인 후 RNDIS 사용으로 설정 변경 및 단말 재부팅
- 다시 PC에 연결하여 WCM 정상접속 및  RNDIS 사용설정되있으며, 
  USB 테더링도 정상이용됨 확인
>> USB 테더링 이용 관련 서브기기 WCM 사이트 접속 자체 불가 시 타 서브단말로 
 WCM 접속하여 설정 테스트 진행 부탁드립니다~
