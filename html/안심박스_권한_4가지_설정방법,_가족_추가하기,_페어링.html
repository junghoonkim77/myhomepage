ㅁ어플 : KT 안심박스

ㅁ증상 : 홈 화면으로 이동하거나 전원 off/on 후 안심박스 해제되어 
 열공모드 미작동 시 4가지 권한 확인 필요



▶ KT 안심박스 권한 4가지 설정 방법

① 설정 > 애플리케이션 > 우측 상단 메뉴 > 특별한 접근 > 
다른앱 위에 표시 > KT 안심박스 > 사용 (ON)

② 설정 > 애플리케이션 > 우측 상단 메뉴 > 특별한 접근 > 
디바이스 관리자 앱 > KT 안심박스 > 사용 (ON)

③ 설정 > 애플리케이션 > 우측 상단 메뉴 > 특별한 접근 > 
사용정보 접근 허용 > KT 안심박스 > 사용 (ON)

④ 설정 > 접근성 > 설치된 서비스 > KT 안심박스 > 사용 (ON)

(보호자)
안심박스 앱 실행 -> 보호자 선택 -> 휴대폰 번호 or kt id -> 
사용할 본인 회선 선택 ( kt가입자가 아니면 조회되지 않음) ->닉네임 만든후 끝

(보호대상자 )
안심박스 앱 실행 -> 보호대상자 선택 -> 휴대폰 번호 or kt id -> 
사용할 본인 회선 선택 ->닉네임 만든후 끝

※페어링 방법 ( 보호자와 보호대상자 연결)

자녀 휴대폰에서 앱실행후 가족연결 페이지에서 가족연결 하기( 보호자 추가하기 ) ->
자녀폰에서 보호자 등록요청시 화면에 6자리 페어링 코드 확인됨 ->  
부모폰에서 (보호대상자 추가하기) 등록에서 자녀회선 번호 확인후 
자녀폰에 표시된 페어링 코드 확인후 입력하면 페어링 완료됨
