최근 삼성폰 OS 업데이트 이후 일부 단말기에 투폰 이용불가 
인입이 적지만 꾸준히 지속되고 있습니다.

"VOC 증상 : 투폰계정생성 불가, 투폰 문자 수신지연 등"
이와 관련 KT 내 단말기술팀에서도 뚜렷한 원인 확인이 불가하여 
최소한의 대응안 공유드립니다.

아래 3가지 응대안들은 모두 진행하시는것이 아니라 상황에 맞게 
한가지 정도만 진행해주시면 됩니다.

※ 투폰SMS 재전송 등 기본적인 조치방법으로 해소 불가시

ㅁ투폰 강제 활성화 앱 설치 안내
1. 활성화 앱 다운링크 복사 후 고객에게 문자 전송  
(링크주소 : 
 https://drive.google.com/file/d/17gZbydGSGPFxeJ6htseTC9brZZJRGyEW/view?usp=drivesdk 
 ) - 링크 오류시 제보주세요

2. 고객 폰에서 ktServiceMenuSupport.APK 해당파일 다운 후 설치 
(알수 없는 앱 설치허용 필요)

3. 원폰모드에서 실행 후 계정생성여부 [미가입]의 “변경” 
버튼 선택하여 [가입]상태로 전환시 해소
- 위 방법으로 해소불가시 (투폰 재가입 필요) 
★4, 5번 방법은 번호재사용 불가로 안내 시 주의가 필요합니다. 진행안해도 무방

4. 설정>고객 특화 서비스>KT 투폰서비스>KT 투폰계정관리> 투폰삭제

5. 휴대폰 상태바에서 투폰모드 ② 표시 삭제된 것 확인 후 투폰 재가입

→ 강제 활성화 방법으로도 해소가 불가하거나, 추후 동일 현상 
발생시에는 삼성AS센터 방문 안내가 필요합니다.
