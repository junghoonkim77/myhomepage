1. Web 설정 화면의 '단말기 관리 > RNDIS' 메뉴에서 RNDIS 사용 여부를 '사용'으로 설정합니다. 
   *설정 변경 적용을 위해서는 제품의 재부팅이 필요합니다.
2. 본 제품을 동봉된 Micro USB 케이블로 PC에 연결합니다. 
3. 처음 사용 시 USB와 연결된 PC에 자동으로 USB Driver가 설치가 됩니다. 
드라이버가 자동으로 설치된 후 PC와 USB 연결을 분리한 후 다시 연결하여 사용하면 됩니다.  
4. 웹브라우저를 실행하여 정상적으로 인터넷이 연결되는지 확인하세요.

※ 본 네트워크 연결은 Windows 8 기준으로 설명되어 있습니다. 그 외 기기의 경우
 제조사마다 연결방식이 다르므로 각 기기의 설정에 
 따라 연결하시기 바랍니다.

※ PC에서 IP주소가 고정으로 설정된 경우에는 통신이 정상적으로 되지 않을 수 있습니다.
※ 본 기능을 사용하지 않고 PC에 연결하여 충전만 하고자 하는 경우에는 RNDIS 사용 여부를 
'미사용'으로 설정하여 주세요.
※ Windows XP/Vista/7 의 경우는 별도 드라이버 설치가 필요합니다.
