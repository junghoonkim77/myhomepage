 K-지존 지존 확인 시엔 앞자리 , 중간자리 , 끝자리에 예를 들어 060 번호가 
입력되면 060이 포함된 모든 번호가 차단되는 내용으로 조회됩니다만 
좀 더 자세한 설명이 필요해 보여 아래 내용 공유합니다.
  
 
예시 번호 070-1234-5678 

(앞자리 차단 방식 ) 
070을 입력하면 070 전체 번호 차단 , 0701을 입력하면 070-1로 시작되는 번호만 차단 
070-12로 입력하면 070-12까지 조건이 맞는 번호만 차단되며 위 조건과 
맞지 않는 070 번호에서 걸려오는 다른 번호들은 모두 수신이 됩니다. 
( 070-123 , 070-13 등은 차단되지 않음)
  

(뒷자리 차단 방식) 
678을 차단하고 싶을 땐 678만 입력 시 차단되며 , 
5678을 입력하면 5678 번호와 일치하는 번호 
만 차단되며 4678,3678등은 차단되지 않습니다.

(중간자리 차단 역시 동일)

즉 포괄적인 차단을 원하는 고객이라면 차단 번호를 적게 입력해야 됩니다.
 
 EX. (앞자리 차단 기준) 

070 번호 전체 차단 요청 -> 070 번호만 입력 
070-1000번대 모두 차단요청 시 -> 0701 입력 
(070 일괄 차단은 안되며 070-1000번 대만 차단됨 )

070-1200번대 모두 차단요청 시 -> 07012 입력 
( 070-1000번대 일괄차단은 안되며 1200번 대만 차단됨) 

070-1230번대 모두 차단 요청 시-> 070123 입력 
( 070-1200번대 일괄 차단은 안되며 1210번 대만 차단됨)

세분화되며 세분화된 조건에 맞는 번호만 차단됨으로 번호 입력 시 주의 필요합니다
