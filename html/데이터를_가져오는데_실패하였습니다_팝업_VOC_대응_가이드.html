[원인]  절전 모드 설정된 3rd party App이 Background에서 
데이터 접속 시도 하거나 BackgroundàForeground로 올라올 때 Rule이 
적용되는 시간을 고려하지 않고 설계된 경우 팝업 발생

[고객대응]
  1. 어떤 app에서 알림을 보낸 건지 확인하는 방법 : Galaxy Store에서 
   Good Lock을 설치하고 실행
    1) 화면 하단 Family 탭에서 Nice Catch를 선택하여 설치
    2) 네트워크를 사용할 수 없다는 토스트 알림이 발생하면 Nice Catch를 실행
    3) 토스트 알리미 선택하여 확인

2. 조치 방법
   1) 사용하지 않는 app이면 삭제
   2) 배터리 사용량 최적화 설정에서 해당 app이 background에서 
    internet을 사용할 수 있도록 예외 처리(아래 3.참고)

3. 예외 처리 설정방법 : 화이트 리스트 추가
   1) 백그라운드 데이터 사용 허용 & 데이터 세이버 미적용앱으로 설정
  -  설정 > 애플리케이션 > 원하는 앱선택 > 모바일 데이터 > 데이터 사용 > 
    백그라운드 데이터 사용 허용 > ON
  - 설정 > 애플리케이션 > 원하는 앱선택 > 모바일 데이터 > 데이터 사용 > 
   데이터 절약모드 > 사용안함 or 데이터 절약 모드 사용 중일때 데이터 사용 허용됨 앱 설정
   2) 배터리 최적화 앱 제외
  - (S OS 이전)설정 > 애플리케이션 > 원하는 앱선택 > 배터리 > 
   배터리 사용량 최적화 > 모든 어플리케이션 > 필요 앱 OFF (파랑->회색)
  - (S OS 이후)설정 > 애플리케이션 > 원하는 앱선택 > 배터리 > 제한없음 선택
