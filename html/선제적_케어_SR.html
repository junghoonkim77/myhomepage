■ SR 수집 기준 FAQ
TT회신과 > 선제적케어
3개월이내 TT발행건이 있으면 TT회신으로 발행, 
TT발행건이 없고 품질불량관련 VOC가 접수되면 선제적 케어 발행

1. 캠페인 이력에는 내용이 남아있지 않습니다. 무조건 3개월 이내 
 재인입 횟수 상관없이 모든 품질 관련 SR은 선제적케어 인가요? 
    아니라면, 위 내용처럼 팝업이 되는 고객들만 선제적 케어인가요?
=> 팝업 고객만이 대상입니다. 캠페인 이력을 별도로 확인하실 필요는 없습니다.
=> 혹시 KOS구현이 잘못 되어 있을 수 있으니 선제적 캠페인이력이 없는데 
 Pop-up이 발생하는 경우가 있다면 Sample을 알려주세요!

2. 모든 품질 SR은 선제적 케어인건가요? (단말기1이나 단말기2등 증상 해소된 경우도 포함)
=> 모든 건이 선제적 케어는 아니고 품질불량 관련 VOC의 경우만 선제적케어로 하시면 됩니다. 
즉 장애성으로 일시적인 현상이나 고객원인등은 기존 분류방식으로 분류하시면 됩니다.
접수내용중 선제적커어로 남겨야 하는 4차 업무유형은 대략 
단말기1, LTE커버리지, 시스템1(무선망), 시스템2(Core망), 중계기설치요청, 
지역무관, 커버리지(특정지역), 접속불가/끊김 등등 입니다. 

3. 특정장소나 지역무관, 중계기점검 상관없이 TT 발행건만 선제적케어로 남겨야 하는지요?
=> 2번 답변과 같이 품질 불량관련 TT발행건에 대해서만 선제적케어이구요. 
 다른 사유의 TT건은 기존처럼 남기시면 됩니다. 

4. 특정장소만 선제적케어로 남긴다면 TT발행없이 양해종결건도 선제적케어인지요?
=> 특정장소는 선제적케어가 맞구요..양해종결건도 품질불량 관련 내용이면 
 선제적케어가 맞습니다. 

5. 선제적케어로 TT 발행 후 재인입되어 다시 TT 발행하면 또 선제적케어로 해야하는지요?
=> 해당 케이스가 TT회신 조건이면 TT회신으로 남기시고 아니면 선제적케어로 하면 됩니다. 

6. 선제적케어로 TT 발행후 재인입되어 그냥 종결건도 선제적케어 인지요?
=> 해당 케이스가 TT회신 조건이면 TT회신으로 남기시고 아니면 선제적케어로 하면 됩니다.

7. 선제적케어로 TT 접수 후 중계기 설치후 3개월 이내 재인입되어 점검 요청시
 TT회신인가요? 선제적케어 인가요? 중계기 점검 인가요?
=> 중계기 점검으로 하시면 됩니다. 

8. 선제적케어 TT접수 후 부재 로 재발행시 TT회신인가요? 선제적케어 인가요?
=> TT회신으로 하시면 됩니다. 

9. 선제적케어 TT 접수 후 물량부족이나 개선불가로 양해 종결 후 재인입하여 
 진행사항을 문의할 경우 TT회신인가요? 선제적케어 인가요?
=> TT회신으로 하시면 됩니다
