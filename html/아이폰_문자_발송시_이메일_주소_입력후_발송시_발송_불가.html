■ 문의 : 아이폰 문자 발송시 이메일 주소 입력후 발송시 발송 불가
[로그 없음]

■ 확인 및 처리내용 :  
- 고객이 발송하려는 이메일 주소가 고객 아이폰 계정으로 등록되어 있으며 
 단말 內 아이메시지 활성화 상태 확인

- 활성화 상태에서 메시지쪽에서 이메일주소 입력 후  발송하면 
 아이메시지 형태로 문자 발송 ( 이메일 확인시 수신기록없음 )

- 아이메시지 비활성화 및 기존에 이메일주소로 문자 발송한 기록 삭제후 
 테스트시 MMS형태로 변환되어 문자 발송되며 이메일 확인시 수신됨확인 

 ＃ 즉, 고객이 사용하는 이메일주소가 아이폰 계정이고 아이메시지 활성화상태면 
 아이메시지 비활성화후 이용 권유 안내 하시면 됩니다.
