ㅇ 고객 문의 : 모바일 데이터 접속불가/속도느림
1. 모바일 데이터 접속불가/속도느림 문의 시 QOS 적용 안내

→ 품질>데이터서비스>고객원인 / 상품코드 : 모바일(요금제)


2. 모바일 데이터 접속불가/속도느림 문의 시 와이파이 ON → OFF로 상태 변경 후 가능 안내

→ 품질>데이터서비스>고객원인 / 상품코드 : 모바일(요금제)
- 고객이 와이파이에 대한 인지가 없이 데이터 이용불가 문의 상태에서 단말기 설정만 끄고 정상 확인 후 종결된 경우


3. 모바일 데이터 접속불가/속도느림을 탐색 도중 '와이파이 접속불가/속도느림'으로 탐색 또는

'와이파이 접속불가/속도느림'을 고객이 직접 언급한 경우 (단, 공용 와이파이는 제외)

→ 품질 > 데이터서비스 > 홈/사설/타사 AP(WIFI)불만 / 상품코드 : 모바일(요금제)

- 모바일 데이터 접속불가/지연으로 인입해서 상담 중에 고객이 와이파이도 안되었다 언급한 경우
