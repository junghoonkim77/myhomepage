특정갤럭시 단말기 발신번호 숨김 에러

ㅇ 현상: 발신번호 표시제한 부가서비스 비활성화 상태임에도, 발신번호 표시제한됨

ㅇ 분석내용: 원인은 알 수 없으나, #31# (피쳐코드)를 통한 발신번호 표시제한 설정이 남아 있음

ㅇ 조치방법: 아래 방법대로 진행 or 삼성 서비스센터 안내
 
 1) 보안 위험 자동차단 off 시킴 (경로 : 설정 → 보안 및 개인정보 보호 → 보안위험 자동차단)
 2) 삼성키패드에서 319712358 을 눌러 히든 메뉴 진입 (KT 패스워드 : 774632)
     → Network Setting > Network mode > 네트워크 설정을 WCDMA ALL 선택
 3) 삼성키패드에서 #31# 누르고 통화 발신버튼 누름 
 4) 히든 메뉴 재진입 후 Automatic으로 변경, LTE 또는 5G 상태에서 동일 문제 발생 여부 재확인
