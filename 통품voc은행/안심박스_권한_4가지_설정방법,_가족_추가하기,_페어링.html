안심박스 앱을 정상적으로 이용하려면 
(접근성 ,다른 앱 위에 표시,디바이스 관리자 앱 ,사용정보 접근 허용) 
필수 권한 4가지가 반드시 ON 으로 설정이 되어야 합니다. 

먼저 

보호자의 KT 안심박스 앱 → 화면 하단 [보호 가이드] 
→ [권한 설정 일시 해제]의 우측토글을 이용하여 해당기능 'ON' 설정하시면 
보호 대상자(자녀) 단말기에서 설정> 필수 권한 변경 화면 진입이 가능합니다.  
단
KT 안심박스의 필수권한이 수동으로 해제되지 않도록 
평상시에는 보호자 앱에서 [권한 설정 일시 해제]를 ‘OFF’로 설정 부탁드립니다.

위 조치후 아래 내용대로 진행

보호대상자님(자녀)의 단말기를 재부팅 후 KT안심박스 앱의 
로그인 및 필수 권한 설정 상태를 확인 한 후 KT안심박스 앱을 실행부탁드립니다.
4가지 필수 권한 설정 경로를 안내해드리오니 참고 부탁드립니다.
a. 설정 → 접근성 → 설치된 앱 → KT 안심박스 →반드시 해제 후 사용(On) 및 KT 안심박스 바로가기 텍스트 클릭
1.측면 버튼과 음량 올리기 버튼 누르기 :사용
2.접근성 버튼 누르기:사용안함

b. 설정 → 애플리케이션 → 우측 상단 더보기(︙) → 특별한 접근 → 다른 앱 위에 표시 → KT 안심박스 사용
c. 설정 → 애플리케이션 → 우측 상단 더보기(︙) → 특별한 접근 → 기기 관리자 앱 → KT 안심박스 사용
d. 설정 → 애플리케이션 → 우측 상단 더보기(︙) → 특별한 접근 → 사용정보 접근 허용 → KT 안심박스 > 사용 

 

KT 안심박스는 네트워트 기반 서비스이므로 보호 대상자(자녀) 단말기의 네트워크 상태가 
불안정할 경우 정책 반영에 수 분이 소요되거나, 정상적으로 반영되지 않을 수 있습니다.
단말기의 모바일데이터(LTE 등) 또는 WiFi의 ON 여부와 실제 네트워크 상태는 상이할 수 있으므로 
인터넷 등에서 네트워크 통신이 가능한지 확인 부탁드립니다.
