■ 문의 :  발신정지된 단말로 등기문자 발송시 지원되지 않는 단말로 표시

■ 확인 및 처리내용 :  
어전트 답변
발신 단말에 해당 사항에 대한 플로우
발신 단말 -> SMC -> SMS VAS -> IMP(이 부분에서 단말 정보를 카프리에 요청) 
-> SMS VAS -> SMC -> 발신 단말  
IMP에서 카프리에 모델명을 요청 시에 모델명을 주지 않았습니다. 
IMP는 모델명에 따라서 지원 여부를 확인 합니다. 
카프리 확인 결과 일시정지인 단말의 경우 단말명을 주지 않았습니다.  

IMP에서 단말명을 알 수 없어서 단말의 등기문자 지원 여부를 알 수가 없었고, 
따라서 읽음확인을 지원하지 않는다는 메시지를 발송 하였습니다.
